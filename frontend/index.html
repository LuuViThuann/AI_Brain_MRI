      About NeuroScan AI

      A fully software-based MRI brain tumor detection system combining U-Net CNN segmentation with Groq AI-powered medical report generation and interactive 3D brain visualization.

      How It Works

      01
      Upload

      Upload a brain MRI scan (PNG/JPG)

      02
      CNN Prediction

      U-Net model segments tumor regions

      03
      Groq AI Report

      LLM generates medical findings

      04
      3D Visualization

      Tumor mapped onto interactive 3D brain

      Dataset

      Source: Kaggle — LGG MRI Segmentation (mateuszbuda)
      Samples: 3,064 paired images (original + mask)
      Patients: 66
      Tumor Type: Low-Grade Glioma (LGG)

      Tech Stack

      Python · FastAPI · TensorFlow (U-Net) · Groq (llama3-70b) · Three.js · GLTF Models · Vanilla JS

      3D Brain Model

      Format: GLB/GLTF with automatic fallback
      Rendering: PBR materials with real-time lighting
      Features: Tumor highlighting, slice view, interactive rotation

      ⚠ This system is for research and educational purposes only. It is NOT a substitute for professional medical diagnosis. Always consult a qualified neurologist.
